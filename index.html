Введение

Добро пожаловать в Volleyball Match — мобильное приложение-табло для отображения текущего счёта волейбольных матчей. Приложение разработано как одиночный инструмент для спортивных мероприятий и не требует создания аккаунта или подключения к интернету.

Настоящая Политика конфиденциальности описывает, какие данные собирает (или не собирает) приложение и как они используются. Пожалуйста, прочитайте её внимательно.

🚫

Мы не собираем ваши данные

✅

Volleyball Match не собирает, не передаёт и не хранит никаких персональных данных пользователей. Мы не ведём аналитику, не используем сторонние SDK для отслеживания и не подключаемся к внешним серверам.

Все пользовательские настройки (например, счёт, настройки матча) хранятся исключительно локально на вашем устройстве с помощью стандартного механизма Shared Preferences и никуда не передаются.

🔑

Разрешения приложения

Для работы некоторых функций приложению требуются следующие разрешения устройства:

  • 📷
    Камера
    Используется только для сканирования QR-кода при подключении другого устройства в качестве дисплея-трансляции по локальной сети. Видео с камеры не записывается и не передаётся.
  • 📡
    Доступ к локальной сети (Wi-Fi)
    Используется для функции трансляции счёта: приложение создаёт локальный WebSocket-сервер, к которому может подключиться другое устройство в той же сети. Соединение работает только в пределах вашей локальной сети; данные не выходят в интернет.
  • 💡
    Запрет перехода в спящий режим (Wakelock)
    Удерживает экран устройства включённым во время матча, чтобы табло всегда было видно.
  • 🔊
    Аудио
    Приложение воспроизводит локальные звуковые файлы (звук свистка) с устройства. Микрофон не используется.
🌐

Сетевая активность

Приложение не подключается к интернету и не взаимодействует с удалёнными серверами. Единственная сетевая активность — это локальный WebSocket-сервер, который создаётся только при активации функции «Трансляция» и работает исключительно в вашей домашней или офисной сети (LAN/Wi-Fi).

Обнаружение устройств в локальной сети осуществляется через протокол mDNS/Bonjour (библиотека Bonsoir). Никакие данные при этом не покидают локальную сеть.

👶

Дети

Приложение не предназначено специально для детей, однако может использоваться на спортивных мероприятиях в любом возрасте. Поскольку приложение не собирает никаких данных, особые меры защиты детских данных (COPPA / GDPR-K) не требуются.

Introduction

Welcome to Volleyball Match — a mobile scoreboard application for displaying the current score of volleyball games. The app is designed as a standalone tool for sporting events and does not require account creation or an internet connection.

This Privacy Policy describes what data the application collects (or does not collect) and how it is used. Please read it carefully.

🚫

We do not collect your data

✅

Volleyball Match does not collect, transmit, or store any personal user data. We do not use analytics, third-party tracking SDKs, or connect to external servers.

All user preferences (such as scores and match settings) are stored solely on your device using the standard Shared Preferences mechanism and are never sent anywhere.

🔑

App Permissions

Certain app features require the following device permissions:

  • 📷
    Camera
    Used only to scan QR codes when connecting another device as a broadcast display over the local network. Camera footage is never recorded or transmitted.
  • 📡
    Local Network Access (Wi-Fi)
    Used for the score broadcast feature: the app creates a local WebSocket server to which another device on the same network can connect. The connection operates only within your local network; no data is sent over the internet.
  • 💡
    Keep Screen On (Wakelock)
    Keeps the device screen on during a match so the scoreboard is always visible.
  • 🔊
    Audio
    The app plays local audio files (whistle sounds) stored on the device. The microphone is never used.
🌐

Network Activity

The app does not connect to the internet or interact with remote servers. The only network activity is a local WebSocket server that is created only when the "Broadcast" feature is activated and operates solely within your home or office network (LAN/Wi-Fi).

Device discovery on the local network is performed via mDNS/Bonjour (Bonsoir library). No data leaves the local network at any time.

👶

Children

The app is not specifically targeted at children, but may be used at sporting events by people of all ages. Because the app collects no data, no special child data protections (COPPA / GDPR-K) are required.

📝

If the changes are significant, we will publish a notice in the app description on the App Store and Google Play.
